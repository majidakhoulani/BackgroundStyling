Backgroung Image, let's GO!

Properties

There are afew different backround prperties:

background-image

This property is the location of the image itself,for example

h1 { backround-image:url('http/mywebsite.com/img/background.jpg'); }
or
h1 { backround-image:url('images/background.jpg'); }

background-color

This allows us to add color to background, An example in css would like this:

h1 { backround-color:green; }

background-repeat

This is how the image will repeat.The value can be any of the following

background-repeat: repeat;

This repeate on bot X and Y axis, and will essentially title the image across the entire bacground

background-repeat: repeat-x;

This repeats the background on the X axis ,or horizontally

background-repeat: repeat-y;

This repeats the background on the Y axis ,or vertocally

background-repeat: no-repeat;

This shows image only once ,without repeating

background-position

This is actual position of the background image within it's container, The value can be:

background-position:top; background-position: right; background-position: bottom; background-position: left; background-position: center;

it can also be a percentage ,lenght,or combination ,like so

background-position: 50px 20%;

Shorthand!

There is a quick way around background images, and you don't necessary have to type out all of the above code to set up your backdround images the way you want them

background

The 'background' Shorthand can handle them in one!

backround: green url('images/background.jpg') top left no-repeat;
